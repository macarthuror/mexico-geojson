## Sobre el Proyecto

<!-- [![Product Name Screen Shot][product-screenshot]](https://example.com) -->

Se busca tener los GeoJSON de todos los estados junto con todas sus colonias para su uso en analisis de datos principalmente.

Los datos son sacadodos de los [datos publicos](https://datos.gob.mx/busca/dataset/ubicacion-de-codigos-postales-en-mexico) de correos de M√©xico; estos se descargan en formato `.kml` y posteriormente son convertidos a GeoJSON.

<br>

### **Herramientas Utilizadas**

* [MyGeodata](https://mygeodata.cloud/converter/kml-to-geojson)
